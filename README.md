# Capistrano::Twingly

Capistrano 3 tasks used for Twingly's Ruby deployment

## Installation

Add this line to your application's Gemfile:

    gem 'capistrano-twingly', git: 'git@github.com:twingly/capistrano-twingly'

And then execute:

    $ bundle

## Usage

### Nginx tasks

    # Capfile
    require 'capistrano/twingly/nginx'
